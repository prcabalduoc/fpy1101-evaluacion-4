# fpy1101-evaluacion-4

## Instrucciones del programa

Este programa simula un sistema de gestión de libros para una biblioteca.

### ¿Cómo funciona?

Al iniciar, el programa muestra un menú con estas opciones:

1. Agregar libro
2. Buscar libro
3. Eliminar libro
4. Actualizar disponibilidad
5. Mostrar libros
6. Salir

### Requisitos

- Python 3 instalado en el sistema.
- Ejecutar el archivo principal desde la carpeta del proyecto.

### Ejecución

En la terminal, escribe:

```bash
python3 ejercicio1.py
```

### Funcionalidades principales

- Agregar un libro con título, copias y periodo de préstamo.
- Validar que los datos ingresados sean correctos.
- Buscar un libro por título.
- Eliminar un libro registrado.
- Actualizar el estado de disponibilidad según las copias.
- Mostrar todos los libros registrados.

### Notas

- El campo de disponibilidad se asigna automáticamente por el sistema.
- El programa sigue mostrando el menú hasta que el usuario elige la opción 6 para salir.
